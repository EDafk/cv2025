My Projects

Login and Register form-blog app

This is one of my first and most complex applications that I made during study time. In this app made with the Javascript, upon opening the link user have an option to register new account and login with existing account. As the app is not connected with the static database User data is saved only during the session so after closing the browser all data will be lost.
After succesfull registration and login process, user can write a blogs that will be visible to all users. The app is made with the plain Javascript, HTML and CSS. The idea was to make a simple app that will have a login and register functionality where user can write a blog.

Further idea is to make user levels where user can have a different access to the app. Head Admin will be able to delete and edit all blogs, Admin will be able to delete and edit only his blogs, while User will be able only to write a blog.
There is also a search option where user can search blogs by the author name but this option is not functional yet
You can try the app by clicking to this link: Login and Register app.
